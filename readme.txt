== Description ==

Inspired by Yelp komplimente system BP Komplimente is a BuddyPress addon that allows members to send each other komplimente or eGifts. This plugin allows you to connect BoniPress with BP Komplimente and award users points for giving or receiving komplimente.

= Built-in Support =
Here are the most notable features of BoniPress for BP Komplimente plugin.

* Award points for giving komplimente.
* Award points for receiving komplimente.
* Badge support.
* Translation ready.

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type BoniPress for BP Komplimente
3. Find BoniPress for BP Komplimente and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
